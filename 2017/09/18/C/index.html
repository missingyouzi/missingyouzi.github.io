getchar()返回值为int
所以以下代码会出现问题

#include
int main(void)
{
char ch;
while(ch = getchar() != ‘#’)
putchar();
return 0;
}
正确代码应该是:

#include
int main(void)
{
char ch;
while((ch = getchar()) != ‘#’)
putchar();
return 0;
}
