JAVA语言

JAVA的特点:代码往往很长, 但是由于其强类型、静态解释型语言的特点, 鲁棒性较好, 安全, 高并发, 多线程, 跨平台, 完全面向对象, 简单(较C++而言), 且JAVA类库较多, 开发可不用重复造轮子

开发工具选择:Eclipse或者IDEA(经过学生认证后可免费使用JetBrains全套软件, 包括进行python开发的pycharm,且现在java开发人员正在逐渐转向IDEA)

运行环境:使用JavaSE或者JavaEE

java 基础

数据类型:

常见修饰符 内存占用 特点
boolean 2字节 只有2个值:true、false, 如果没有赋值, 则默认给 false
byte、short、int、long 1、2、4、8 字节 如果没有赋值, 则默认给 0
char 2字节 以整数形式储存, 转义字符
float、double float占用 4 字节, double占用 8 字节 书写单精度的时候数据最后必须加上f, 否则代表double类型
final 常量声明 如final float PI = 3.14;
类型转换(强制转换、自动转换) (type)data 自动转换byte->short->int->long->float->double由低精度转换为高精度属于自动转换

运算符

赋值运算符:”=”, 如int y = 1;

算术运算符:”+”, “-“, ““, “/“, “++”, “–” *注意

关系运算符:>, <, >=, <=, ==, !=

逻辑运算符:&&, ||, !

条件运算符: ?, :

位运算符: ~(非), &(与), |(或), >>(右移), <<(左移)

其他运算符: . 用来访问对象的成员变量或者方法, new 创建一个对象或者数组
instanceof : object instanceof type_name
如果 object 是 type_name 类创建的对象时, 该表达式的结果是true, 否则是false

运算符 运算符优先级 运算符分类 结合方向
, ; [] () 1 分隔符

&& 11 逻辑或运算 左向右
¦¦ 12 逻辑与运算 左向右
? : 13 三目运算符 左向右
