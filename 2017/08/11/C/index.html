首先，十分感谢阮**小姐姐，让我知道hexo能够搭建博客（差点入了django大坑，虽然以后也会入的((摊手))

现在趁着我还记得一点使用hexo和github的坑，请参照2个网页：

hexo入门指南
http://www.maintao.com/2014/hexo-beginner%27s-guide/

手把手教你用Hexo+Github 搭建属于自己的博客 - gdutxiaoxu的博客 - CSDN博客
http://blog.csdn.net/gdutxiaoxu/article/details/53576018

其次，首先创建一个hexo文件夹，在npm install hexo-cli -g 和 npm install hexo –save 之后请重新创建一个文件夹用于hexo的初始化，之后可以把第一个文件夹里的东西剪切过来

然后在配置sshkey的时候注意点就好了
